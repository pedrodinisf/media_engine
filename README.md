# media_engine

Universal media-processing engine. Typed artifacts, composable operations,
pluggable backends, content-addressed caching, async DAG execution. Ships as
a Python package + CLI (`med`) + daemon + REST + MCP.

## Quickstart (Phase 0)

```bash
uv sync
uv run pytest -q
```
